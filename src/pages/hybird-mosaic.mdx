# mozaïeken maken

Basisstappen om een mozaïek te maken:
1.	Kies een ondergrond voor je mozaïek. Dit kan het beschikbare MDF (20 x 15 cm) zijn, of je kunt zelf een ondergrond maken van restmateriaal in het Makers Lab. Het kan ook een fles, een beker, een notitieboekje, een telefoonhoesje, etc. zijn.
2.	Maak een ontwerp voor je mozaïek op de ondergrond.
3.	Selecteer tesserae om de ondergrond te bedekken. Gebruik verschillende materialen.
4.	Snijd en lijm de gekozen tesserae/stukjes.
5.	Laat het drogen.
6.	De voeg wordt pas aangebracht als de lijm volledig droog is. Dit zullen we waarschijnlijk in de volgende les doen.
